# parsing_avito

Скрипт выполняет несколько функций:
1. Парсит данные с авито
2. Распознавание телефона из объявления

# Подготовка рабочей среды
Перейдите в свою рабочую директорию и выполните следующие команды:
```
git clone https://github.com/avcherezov/parsing_avito.git
cd parsing_avito
```
Создайте и активируйте виртуальное окружение:
```
source -m venv venv
source venv/Scripts/activate
```
Установите необходимые зависимости:
```
pip install -r requirements.txt
```
# Подробнее о функционале
## Парсинг данных с авито
```
при запуске файла avito.py
```
1) скрипт обращается к определенному запросу, получает кол-во страниц
2) циклом проходится по всем страницам и получает следующие данные:
   название, стоимость, местонахождение, ссылку на объявление
3) сохраняет результат в файле avito.csv

## Распознавание телефона из объявления
```
при запуске файла avito_tel.py
```
1) скрипт обращается к определенному объявлению, для этого запускается Chromedriver
2) нажимает на кнопку с телефоном и делает скриншот
3) обрезает картинку по телефону, для этого получает начальные координаты картинки с телефоном и её размер
4) переводит телефон с картинки в текст и выводит на печать
